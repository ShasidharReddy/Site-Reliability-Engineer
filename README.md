# Site Reliability Engineer - Course Kit

Comprehensive SRE training material: theory, hands-on labs, and ready-to-execute artifacts focused on monitoring, observability, Grafana, Kubernetes, and hybrid (on-prem + GCP) operations.

Contents:
- THEORY.md — Concepts, best practices, incident response, SLIs/SLOs, runbooks
- LABS/ — Hands-on labs with step-by-step commands (Minikube/Kind, Prometheus, Grafana, Alerting, SLOs)
- LABS/deploy-prometheus-grafana.sh — Quick deploy script

Usage: clone the repo, follow LABS/README.md to run labs locally or on a test cluster.
