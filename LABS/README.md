Labs - Quick start

1) 01-setup-minikube.md - Local Kubernetes cluster setup (minikube/kind)
2) 02-deploy-prometheus-grafana.sh - Script to deploy kube-prometheus-stack via Helm
3) 03-create-slos.md - Create SLIs/SLOs and alerts using Prometheus rules

5) 05-incident-playbook.md - Hands-on SEV runbook exercises
